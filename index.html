Projects

Calculator

My calculator project is an interactive web application that allows users to perform basic arithmetic operations. Built with HTML, CSS, and JavaScript, it features a responsive keypad with buttons for numbers and functions. The project evaluates expressions in real-time, providing instant feedback on calculations, and demonstrates my coding skills while enhancing my portfolio.

BMI Calculator

My BMI calculator is an interactive web app that helps users calculate their Body Mass Index (BMI). Using JavaScript, it takes weight and height inputs through sliders that update in real-time. The project uses formulas and conditional logic for feedback based on BMI ratings, showcasing my programming skills and adding value to my portfolio.

OTP Verification

My OTP verification project is an interactive web application designed to enhance user security during login. Using HTML, CSS, and JavaScript, it allows users to input a four-digit OTP through individual text fields. The application features automatic focus shift to the next input for seamless user experience. This project demonstrates my ability to create user-friendly interfaces and strengthens my portfolio.

Portfolio Website

My portfolio website is an interactive platform showcasing my skills and projects. Using HTML, CSS, and JavaScript, it features sections for home, about, services, projects, and contact. The design is responsive, ensuring a seamless experience on all devices, and includes a mobile menu for easy navigation, highlighting my web development capabilities.
